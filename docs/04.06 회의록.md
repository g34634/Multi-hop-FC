            - 주요 기능 소개
                
                **1. 프로그램 기반 추론 (Program-Guided Reasoning)**
                
                단순한 키워드 검색을 넘어, LLM(**Llama3, DeepSeek-R1**)이 질문을 분석하여 논리적 실행 계획(파이썬 코드)을 수립합니다.
                
                - **Planner**: 질문을 `Question()`, `Verify()`, `Predict()` 함수 단위로 분해.
                - **Executor**: 생성된 프로그램을 실행하며 각 단계에 필요한 최적의 정보를 수집.
                
                **2. 동적 위키피디아 인덱싱 (Dynamic Indexing)**
                
                로컬 DB에 답변에 필요한 정보가 없을 경우, 시스템이 스스로 Wikipedia API를 호출하여 실시간으로 문서를 수집하고 벡터 인덱스에 추가합니다.
                
                - **Self-Correction**: 지식의 부재를 스스로 감지하고 실시간으로 지식 베이스를 확장.
                
                **3. 멀티 모델 비교 시스템**
                
                - **Llama3 (latest)**: 로컬 환경(MacBook Air 등)에서 가장 빠르고 안정적인 논리 설계 및 검색 성능을 제공하는 실전형 모델.
                - **DeepSeek-R1 (8b)**: 심층 사고(`think`) 과정을 통해 복잡한 추론을 수행하는 연구형 모델 지원.
                
                **4. 데이터 정제 및 안정화 (Sanitization)**
                
                - **Unknown Guard**: 정보 부재 시 무의미한 추론 전파를 차단하여 환각(Hallucination) 방지.
                - **String Cleaner**: LLM 답변에서 불필요한 수식어를 제거하여 검색어 순도를 극대화
